# QiCYCLE Tire Pressure Calculator 🎯

An interactive tire pressure calculator and product showcase for [QiCYCLE](https://www.qicycle.com) mini electric tire inflators.

**Live Demo:** https://alwinqin.github.io/qicycle-air-inflator/

## Features

- **Smart Calculator** — Select your vehicle (car, bike, kids bike, sports ball), get the ideal PSI with temperature & weight compensation
- **Interactive Gauge** — Animated SVG gauge with count-up animation and sound effects (PSI-dependent duration and pitch)
- **Product Recommendations** — Intelligent scoring matches the best QiCYCLE inflator to your needs
- **Multi-Language** — English, 中文, Français, Deutsch, 한국어
- **Fun Extras** — "Guess the Pressure" challenge, emoji share text, air pump sound effects

## Usage

1. Open `index.html` in any browser — zero dependencies
2. Click **Get Started**
3. Select what you're inflating → choose details → **Calculate**

Or deploy to GitHub Pages:
```
Settings → Pages → Source: deploy from main branch root
```
